# MarketPulse UI 升级设计文档

## 目标

将 MarketPulse 从功能型界面升级为高级金融感 UI，新增设置页面。

## 视觉体系

### 配色

| 元素 | 深色模式 | 浅色模式 |
|------|---------|---------|
| 背景 | 渐变 #0B0D17 → #101829 | #F2F4F8 |
| 卡片 | rgba(255,255,255,0.06) + 1px渐变边框 | #FFFFFF + 阴影 |
| 涨 | #00E676 渐变 | #4CAF50 |
| 跌 | #FF1744 渐变 | #F44336 |
| 强调色 | #F0B90B (金色) | #D4A017 |
| 主文字 | #E8ECF1 | #1A1A2E |
| 副文字 | #6B7994 | #8E95A4 |

### 卡片样式

- 半透明背景 + 1px 渐变边框（深色模式）
- 白色背景 + 柔和阴影（浅色模式）
- 圆角 20pt continuous

### 动画

- 数字变化：withAnimation(.spring) 过渡
- 列表加载：staggered fade-in
- 下拉刷新：卡片缩放回弹

## 总览页改进

- 顶部市场状态指示条
- 商品卡片加迷你 sparkline 背景
- 板块预览改横向滚动标签

## 设置页面

- 主题切换（深色/浅色/系统）
- 语言切换（中文/English）
- 货币单位（USD/CNY）
- 通知设置（价格提醒开关 + 阈值）
- 关于（版本、数据源、免责声明）

## 不做

- 自定义 watchlist
- K 线图详情页
- 用户登录系统
